유니티의 광고 최적화 플랫폼, 레벨플레이를 활용하여 <br>
쉽고 빠르게 모바일 게임 어플리케이션에 광고를 추가하고 수익화를 배우는 예제 프로젝트입니다.  
  
### 미디에이션이란?  
최적의 광고를 나만의 게임 어플리케이션이 가진 광고 유닛에 노출하여  
수익을 최적화 하는 방법을 의미합니다.  
  
### 장면 소개
장면은 1개로 구성하고 있으며 복셀 그래픽 기반의 간단한 캐주얼 게임 로직을 갖추고 있습니다.  
레벨플레이 SDK 제작 실습은 **AdManager.cs**에서 이루어집니다.  
  
### 학습 방법
**Unite Seoul 2026 Hands-On** 트레이닝에서 오프라인으로 학습이 진행되며  
차후 **Unity Learn** 콘텐츠로 출간 예정입니다.
